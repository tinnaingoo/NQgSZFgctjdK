ကွန်ပျူတာဆိုတာဘာလဲ?

By tinnaingoo • Feb 23, 2025

ကွန်ပျူတာ ဆိုသည်မှာ ဒေတာများ သိမ်းဆည်းရန် နှင့် တွက်ချက်ရန် အသုံးပြုသော လျှပ်စစ်ပစ္စည်းဖြစ်သည်။
စာရွက်စာတမ်းများ နှင့် စာရင်းဇယားများပြုလုပ်ခြင်း၊ email ပို့ခြင်း၊ Game ဆော့ခြင်း၊ Internet အသုံးပြုခြင်း နှင့် အခြားသောလုပ်ငန်းဆောင်တာများတွင် အသုံးပြုကြသည်။ ကွန်ပျူတာအမျိုးအစားများမှာ

Processing နှင့် Storing ဆိုတာ ဘာလဲ?
Intel Core i7 4th Gen
ကွန်ပျူတာ ဦးနှောက် CPU ( Central Processing Unit ) ဖြစ်သည်၊ တစ်နည်းအားဖြင့် Processor ဟုလည်း ခေါ်သည်။
Storage Unit ဆိုသည်မှာ HDD (Hard Disk Drive), SDD (Solid State Drive), USB (Universal Serial Bus) နှင့် အခြားသော ဒေတာများ သိမ်းဆည်းနိုင်သည့် devices များကို ခေါ်သည်။
ကွန်ပျူတာ တစ်လုံး၏ လုပ်ဆောင်ချက်မှာ အခြေခံအားဖြင့် ကျွန်တော်တို့ဟာ Keyboard, Mouse, Microphone များကို အသုံးပြုပြီး ကွန်ပျူတာရဲ့ ဦးနှောက်ခေါ် CPU ကို လုပ်ဆောင်ခိုင်းစေပါတယ်။ CPU က ခိုင်းစေချက်များကို တွက်ချက်ပေးပြီး သောအခါ Output အဖြစ် Monitor, Projector, Speaker, Printer စသည်တို့မှ ပြန်လည်ဖော်ပြပေးပါတယ်။ CPU ကို လူသားတို့ ဦးနှောက်နှင့် နှိုင်းယှဉ်သောအခါ ထူးခြားမှုတစ်ခုရှိပါသည်။ အဲ့တာကတော့ CPU မှာ တွက်ချက်မှု (Processing) ကိုသာ လုပ်ဆောင်ပေးနိုင်ပြီး မှတ်သားမှုများ (Memory) ကို မလုပ်ဆောင်ပေးနိုင်ပါ။ လူသားရဲ့ ဦးနှောက် ကတော့ တွက်ချက်မှုကိုရော မှတ်သားမှုကိုပါ လုပ်ဆောင်ပေးနိုင်ပါသည်။ ထို့ကြောင့် ကွန်ပျူတာ ရဲ့ CPU ကတော့ ကူညီမှတ်သားပေးနိုင်မည့် ပစ္စည်းလိုအပ်လာပါသည်။ သူကတော့ memory ပါ။ ကျွန်တော်တို့ Input ပေးလိုက်တဲ့ ခိုင်းစေချက်တွေဟာ Memory ပေါ်ကို ဦးစွာရောက်ရှိသွားပြီး CPU ကို တွက်ချက်မှုများလုပ်ဆောင်စေဖို့ ပို့ဆောင်ပါတယ်။ CPU မှ တွက်ချက်ပြီးသော data များကို Memory သို့ ပြန်လည်ပို့ဆောင်ပြီး Memory မှ Output Device တွေကို ထပ်မံ ပေးပို့ပါတယ်။
ဗဟုသုတအနေဖြင့် မှတ်သားထားရမှာက CPU က လုံးဝ မှတ်သားထားမှုကို မလုပ်တာတော့ မဟုတ်ပါဖူး။ အရေးကြီးပြီး ခဏခဏ ပြန်လည် အသုံးပြုရမည့် ဒေတာတွေ၊ ခိုင်းစေချက်တွေကိုတော့ သူ့မှာ အနည်းငယ်ပါဝင်တဲ့ Cache Memory ထဲမှာ မှတ်သားပါတယ်။ ဒါကို Level အဆင့်အနေနဲ့ ခွဲပြီး L1 Cache, L2 Cache, L3 Cache ဆိုပြီးမှတ်သားပါတယ်။ L1 ကတော့ cache ဖမ်းတာ အမြန်ဆုံးဖြစ်ပြီး data ပမာဏ အနည်းဆုံးဖြစ်ပါတယ်၊ L1 က Cache ဖမ်းတာ memory ပြည့်သွားရင် L2 ဆီပို့ဆောင်ပါတယ်။ L3 က cache ဖမ်းတာ အမြန်ဆုံးဖြစ်ပါတယ်။
ဒီနေရာမှာ Memory ရဲ့ အားနည်းချက်က Hard Disk ဖြစ်ပေါ်လာဖို့အတွက် အကြောင်းအရာဖြစ်လာပါတယ်။ Memory ရဲ့ မှတ်သားမှုဟာ ယာယီ မှတ်သားမှုသာဖြစ်ပြီး ထာဝရ မသိမ်းဆည်းထားနိုင်ပါ။ Power ရရှိချိန်သာလျှင် မှတ်သားထားနိုင်ပြီး Power မရှိလျှင် မှတ်သားထားသမျှ ဒေတာ အကုန် ပျောက်ဆုံးသွားပါသည်။ ဒါကြောင့် ကွန်ပျူတာ Power ပိတ်ထားလျှင်လည်း ဒေတာတွေမှတ်သားထားပေးနိုင်ဖို့ ပစ္စည်းတစ်ခု လိုအပ်လာပါသည်။ အဲ့တာကတော့ Hard Disk ပဲဖြစ်ပါတယ်။ တစ်နည်းအားဖြင့် ပြောရမယ်ဆိုရင် Storage Device ပါ။ ကွန်ပျူတာမှာ ကျွန်တော်တို့လုပ်နေသမျှ အလုပ်တွေ အားလုံး Save မလုပ်မချင်း Memory ( RAM ) ထဲမှာပဲ ရှိပါသေးတယ်။ သိမ်းဆည်းမှသာ Storage Device ထဲ ရောက်မှာပါ။
Basic Function Of PC
Processor ကတော့ တပ်ဆင်သမျှ Device တွေကို ထိန်းချုပ်ကွပ်ကဲ ရာမှာ အဓိကျ ကျပါတယ်။ အကြမ်းအားဖြင့် လမ်းကြောင်းသုံးခု ရှိပါတယ်။ Data, Address, Control တို့ ဖြစ်ပါတယ်။ Data ကတော့ User ဖန်တီးတည်ဆောက်လိုက်တဲ့ အချက်အလက်တွေပါ။ Address ကတော့ CPU လိုချင်တဲ့ data တွေဟာ memory ရဲ့ ဘယ်နားမှာ ရှိတယ်၊ HDD ရဲ့ ဘယ် location နားမှာ ရှိတယ် ဆိုတဲ့ နေရာလိပ်စာ အတိအကျ ပြောတဲ့ Address လမ်းကြောင်းတွေဖြစ်ပါတယ်။ Control ကတော့ CPU ကတွက်ချက်လို့ ရလာတဲ့ Result တွေ ခိုင်းစေချက်တွေ၊ ညွှန်ကြားချက်တွေ ဖြစ်ပါတယ်။
Input Output Device ( I/O Device) တွေကတော့ ကွန်ပျူတာ မှာ ဒေတာ အသွင်း အထုတ်ပြုလုပ်ပေးတဲ့ ပစ္စည်းများ ( KeyBoard, Mouse, Mouse, Monitor, Printer) တွေကိုခေါ်တာပါ။ I/O Device တွေကို Memory နှင့် CPU မှ Data, Address, Control နည်းလမ်းတွေမှ တစ်ဆင့် ချိတ်ဆက်ဆောင်ရွက်ပါတယ်။
