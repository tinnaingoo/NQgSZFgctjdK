ကွန်ပျူတာဆိုတာဘာလဲ?

By tinnaingoo • Feb 23, 2025

ကွန်ပျူတာ (Computer) ဆိုသည်မှာ ဒေတာများ (data) ကို ထည့်သွင်း (Input)၊ တွက်ချက် (Processing)၊ သိမ်းဆည်း (Storage) နှင့် ထုတ်ပေး (Output) အနေဖြင့် လုပ်ဆောင်နိုင်သော လျှပ်စစ်ပစ္စည်းတစ်ခု ဖြစ်သည်။ သင်၏ Keyboard၊ Mouse စသည့် Input Devices များမှ ကွန်ပျူတာ၏ CPU (Central Processing Unit) သို့ ညွှန်ကြားချက်များ ပို့ဆောင်ပြီး၊ CPU သည် အဆိုပါ ညွှန်ကြားချက်များအရ အချက်အလက်များကို တွက်ချက်၊ ဖော်ထုတ်၊ သိမ်းဆည်း၍ Output Devices (Monitor, Printer, Speaker စသည်) သို့ ပြန်လည်ပို့ဆောင်သည်။

 

ကွန်ပျူတာ၏ အဓိပ္ပာယ်နှင့် လူသားများအတွက် အကျိုးကျေးဇူး

  1. အလုပ်လုပ်ဆောင်မှု တိုးတက်စေခြင်း
    ကွန်ပျူတာများသည် ဒေတာများကို အလွန်မြန်ဆန်စွာ တွက်ချက်ပေးနိုင်၍ စီးပွားရေး၊ ပညာရေးနှင့် အခြားသောလုပ်ငန်းများတွင် အချိန်နှင့် အားကုန်ကျစရိတ်ကို လျော့ချပေးသည်။

  2. အချက်အလက် သိမ်းဆည်းခြင်းနှင့် အချက်အလက်ရှာဖွေမှု
    အင်တာနက်နှင့် Cloud Storage စနစ်များက လူသားများကို အချက်အလက်များကို လွယ်ကူစွာ သိမ်းဆည်း၊ လက်လွှတ်မခံဘဲ မည်သည့်အချိန်မဆို ရယူနိုင်စေရန် ကူညီပေးသည်။

  3. ဆက်သွယ်ရေးနှင့် ပူးပေါင်းဆောင်ရွက်မှု
    Email, Video Conference, Social Media စသည်ဖြင့် ကွန်ပျူတာများသည် လူသားများအကြား အမြန်ဆန်သော ဆက်သွယ်ရေးနှင့် ပူးပေါင်းလုပ်ကိုင်နိုင်စေသည်။

  4. နည်းပညာဆိုင်ရာ ဖန်တီးမှုများ
    Software Development, AI, Machine Learning စသည့် နည်းပညာခေတ်သစ်များကို ဖန်တီးနိုင်စေပြီး လူသားများ၏ ဘဝအဆင်ပြေအောင် တိုးတက်လာစေသည်။

ကွန်ပျူတာ ရဲ့ သမိုင်းနှင့် စတင်တီထွင်ခဲ့သူများ

  1. အစပျိုးစခန်း
    အစောဆုံးကွန်ပျူတာဆိုင်ရာ စိတ်ကူးများသည် လူသားတို့၏ ရှေးဦးတွက်ချက်ပစ္စည်းများ (ဥပမာ: Abacus) မှစ၍ ဖြစ်လာခဲ့သည်။

  2. စက်ပညာရှင်များ၏ တီထွင်မှု

    • Charles Babbage – ၁၈၃၇ ခုနှစ်တွင် Analytical Engine ကို အကြံပြုခဲ့သည်။
    • Ada Lovelace – Babbage ၏ အင်လက်တစ်စင်ဂျင်ကို ပထမဆုံး Programming Concepts အဖြစ် ဖော်ထုတ်ခဲ့သည်။
    • ၂၀ ရာစုအလယ်ပိုင်းတွင် Electronic Computer များ (ENIAC, UNIVAC) တီထွင်ခဲ့ကြသည်။

  3. ကွန်ပျူတာဆင့်ကဲ ပြောင်းလဲလာမှု
    ကွန်ပျူတာများကို ပထမဆုံး ဂဏန်းတွက်ချက်စက် (Mechanical Calculators) မှ စတင်၍ Analog, Digital နှင့် အခုခေတ် AI/Quantum Computer များအထိ ဆင့်ကဲ ပြောင်းလဲလာခဲ့သည်။

        

Computer Generations

 

မြန်မာနိုင်ငံတွင် ကွန်ပျူတာအသုံးပြုမှု

  • စတင်အသုံးပြုမှု
    မြန်မာနိုင်ငံတွင် ကွန်ပျူတာကို ပထမဆုံး အစပိုင်းမှာ တက္ကသိုလ်များတွင် သုတေသနနှင့် ပညာရေးရည်ရွယ်ချက်များအတွက် စတင်အသုံးပြုခဲ့ကြသည်။

  • အများအပြားအသုံးပြုလာခြင်း
    ၁၉၈၀ နှစ်များမှစ၍ အစိုးရ၊ စီးပွားရေးလုပ်ငန်းများနှင့် ပုဂ္ဂလိကကဏ္ဍများတွင် ကွန်ပျူတာ အသုံးပြုမှု ပိုမိုတိုးတက်လာခဲ့ပြီး ယနေ့တွင် လူတိုင်း၏ နေ့စဉ်ဘဝတွင် အရေးကြီးသော အရာတစ်ခုဖြစ်လာသည်။

 

ယနေ့ခေတ် ကွန်ပျူတာအမျိုးအစားနှင့် Brand များ

  1. Desktop Computer

    • အားသာချက်များ: Performance, Upgradeability
    • Brand: Dell, HP, Lenovo

  1. Laptop (Personal Computer)

    • အကျိုးကျေးဇူး: Portability, Compact Design
    • Brand: Apple (MacBook), Dell (XPS Series), HP (Spectre, Envy)

  1. Tablet & 2-in-1 Devices

    • Touchscreen, Versatility
    • Brand: Microsoft Surface, iPad Pro, Samsung Galaxy Tab

  1. Servers & Mainframes

    • High Performance, Large-scale Data Processing
    • Brand: IBM, Oracle, Fujitsu

 

ကွန်ပျူတာတွေက ဘယ်လို အလုပ်လုပ်သလဲ?

ကွန်ပျူတာတစ်လုံး၏ လုပ်ဆောင်ချက်များကို အဓိကအားဖြင့် အောက်ပါ အဆင့်(၄)ခုဖြင့် ဖော်ပြနိုင်ပါသည်။

  1. Input (ဒေတာထည့်သွင်းခြင်း)
    Keyboard, Mouse, Scanner စသည့် Devices များမှ ဒေတာကို ကွန်ပျူတာသို့ ထည့်သွင်းသည်။

  2. Processing (တွက်ချက်ခြင်း)
    CPU (Central Processing Unit) သည် အမိန့်များအရ ဒေတာကို တွက်ချက်ပြီး၊ Logic Operation များကို ဆောင်ရွက်သည်။

  3. Storage (သိမ်းဆည်းခြင်း)
    RAM (Random Access Memory) သည် Temporary Data များကို သိမ်းဆည်းပြီး HDD/SSD သည် အစဉ်အမြဲ သိမ်းဆည်းထားရန် အသုံးပြုသည်။

  4. Output (အချက်အလက် ထုတ်ပေးခြင်း)
    Monitor, Printer, Speaker စသည့် Devices များမှ တစ်ဆင့် လုပ်ဆောင်ချက်ရလဒ်ကို ပြသသည်။

 

ကွန်ပျူတာအတွင်း ပါဝင်သည့် အစိတ်အပိုင်းများ

  1. CPU (Central Processing Unit)
    ကွန်ပျူတာ၏ ဦးနှောက်ဖြစ်ပြီး အမိန့်များကို လုပ်ဆောင်သည်။

  2. Memory (RAM & Cache)
    အချက်အလက်များကို အလျင်အမြန် လက်ရှိတွင် သိမ်းဆည်းထားရန်၊ CPU အမြန်လုပ်ဆောင်မှုအတွက် အရေးကြီးသည်။

        

 

  1. Storage Devices (HDD, SSD)
    အချက်အလက်များကို တာဝန်ခံသိမ်းဆည်းရန်၊ Power မရှိသောအခါပါ သိမ်းဆည်းထားနိုင်ရန်။

        

  1. Motherboard
    ကွန်ပျူတာ၏ အစိတ်အပိုင်းများကို ချိတ်ဆက်ပေးပြီး Communication ချိတ်ဆက်မှုကို စီမံသည်။

        

  1. Power Supply Unit (PSU)
    ကွန်ပျူတာတွင် လိုအပ်သော လျှပ်စစ်ပါဝါများကို ပေးအပ်သည်။

        

  1. Input/Output Interfaces
    USB, HDMI, Audio Jack စသည့် Ports များဖြင့် အခြား Devices များနှင့် ဆက်သွယ်ပေးသည်။

        

ကွန်ပျူတာတစ်လုံး၏ Basic Function များ

ကွန်ပျူတာသည် အခြေခံအားဖြင့် "Input, Process, Store, Output" ဟူသော အလုပ်လုပ်ပုံ (Function) များကို ဆောင်ရွက်ပေးပါသည်။

  • Input: User မှ အချက်အလက်ထည့်သွင်းခြင်း
  • Process: CPU မှ အချက်အလက်ကို တွက်ချက်ခြင်း
  • Store: Memory နှင့် Storage Devices တွင် အချက်အလက် သိမ်းဆည်းခြင်း
  • Output: အချက်အလက်ကို User သို့ ပြသခြင်း

ကွန်ပျူတာ အသုံးပြုသင့်ကြောင်း အကြံပြုချက်

  • အလုပ်နှင့် ပညာရေး
    ကွန်ပျူတာသည် အလုပ်လုပ်ဆောင်မှု အကျိုးကျေးဇူး၊ ပညာရေး လေ့လာမှုနှင့် အချက်အလက် ထိန်းသိမ်းမှုတို့တွင် အရေးကြီးသော ကိရိယာဖြစ်သလို အချိန်နှင့် အရင်းအမြစ်များကို ထိရောက်စွာ စီမံခန့်ခွဲနိုင်စေသည်။

  • စီးပွားရေးနှင့် ဖန်တီးမှု
    Software Development, Data Analysis, Digital Marketing စသည့် နယ်ပယ်များတွင် ကွန်ပျူတာကို အသုံးပြုခြင်းဖြင့် စီးပွားရေးတိုးတက်စေပြီး ဖန်တီးမှုကို တိုးတက်စေသည်။

  • အကျိုးအမြတ်
    ကွန်ပျူတာသည် သင့်ဘဝကို အဆင်ပြေအောင်ပြောင်းလဲပေးပြီး အချက်အလက်များကို လွယ်ကူစွာ ရယူ၊ ပြုပြင်နှင့် သုံးစွဲနိုင်စေသည်။
