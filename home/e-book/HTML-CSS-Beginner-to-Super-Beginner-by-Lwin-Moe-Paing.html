ဒီစာအုပ်က အခုမှ Web Development ကို စတင်လေ့လာမည့်သူ့တွေအတွက် အရမ်းသင့်တော်တဲ့ စာအုပ်တစ်အုပ်ဖြစ်ပါတယ်။ ဒီစာအုပ်မှာ Web Development လုပ်တဲ့အခါမှာ မသိမဖြစ်သိထားသင့်တဲ့ HTML နဲ့ CSS ကို အသေးစိတ်ကအစ ရှင်းပြပေးထားတာမို့ အရမ်းအကျိုးရှိစေမည့် စာအုပ်ဆိုရင်လည်း မမှားပါဘူး။

ဒီ eBook က ဘယ်သူ့အတွက်လဲ?

  • လုံးဝအစပြုသူများ - ကုဒ်တစ်လိုင်းမှ မရေးဖူးသူတွေအတွက်၊ ဒီ eBook က HTML နဲ့ CSS ကို နူးညံ့စွာနဲ့ စနစ်တကျ မိတ်ဆက်ပေးပါလိမ့်မယ်။
  • အလယ်အလတ်သင်ယူသူများ - အရင်က အနည်းငယ် အတွေ့အကြုံရှိပြီး သူတို့ရဲ့ အသိပညာကို ခိုင်မာအောင်လုပ်ချင်သူတွေ၊ ဝက်ဘ်ဖွံ့ဖြိုးရေးမှာ ပိုမိုအဆင့်မြင့်တဲ့ အကြောင်းအရာတွေကို စူးစမ်းလေ့လာလိုသူတွေအတွက်။
  • ဝက်ဘ်ဆိုက်ပိုင်ရှင်များ - သင့်မှာ ဝက်ဘ်ဆိုက်တစ်ခုရှိပြီး ဒီဇိုင်းနဲ့ အသွင်အပြင်ကို ထိန်းချုပ်ချင်တယ်ဆိုရင်၊ ဒီ eBook က သင့်ဝက်ဘ်ဆိုက်ကို သင့်စိတ်တိုင်းကျ ဖြစ်အောင် စွမ်းဆောင်ပေးနိုင်ပါလိမ့်မယ်။

ဒီ eBook ကို ဘာကြောင့် ဖတ်သင့်သလဲ?

  • လက်တွေ့ဆန်တဲ့ ချဉ်းကပ်မှု - သင်ယူထားတာတွေကို ချက်ချင်း အသုံးချနိုင်ဖို့ လက်တွေ့ လေ့ကျင့်ခန်းတွေ ပါဝင်ပါတယ်။
  • ပြည့်စုံတဲ့ အကြောင်းအရာ - အခြေခံကစလို့ တုံ့ပြန်မှုဒီဇိုင်း (Responsive Design) နဲ့ အဆင့်မြင့် နည်းပညာတွေအထိ အကုန်လုံး ဖုံးအုပ်ထားပါတယ်။
  • ကျွမ်းကျင်သူရဲ့ လမ်းညွှန်မှု - ဝက်ဘ်ဖွံ့ဖြိုးရေးသမားနဲ့ ပညာပေးသူအဖြစ် နှစ်ပေါင်းများစွာ အတွေ့အကြုံရှိတဲ့ လွင်မိုးပိုင်က ရေးသားထားပါတယ်။
  • နောက်ဆုံးပေါ် အချက်အလက်များ - အကြောင်းအရာတွေကို နောက်ဆုံးပေါ် ဝက်ဘ်ဖွံ့ဖြိုးရေး အလေ့အကျင့်တွေနဲ့ ခေတ်ရေစီးကြောင်းတွေနဲ့အညီ အမြဲမွမ်းမံထားပါတယ်။
